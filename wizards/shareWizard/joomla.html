If you see a "Plugin Manager" on your Joomla:
  1. Go to plugin manager and click on tiny TinyMCE 2.0
  2. Go to plugin parameters. Select "never" on "Code cleanup" on save option. Then save it.
  3. Find the page you would like to add the form and click on "Edit" icon.
  4. On the editor textarea, click on the "HTML" icon.
  5. On pop-up "HTML Source Editor", copy and paste the code above.
